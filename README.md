# How Vercel Wins With Startups — A Product-Led Strategy

A presentation website built with **Next.js** and deployed on **Vercel**, demonstrating how a Product Manager can go from PowerPoint to production deployment in minutes.

## 🚀 What This Project Demonstrates

- Converting a static slide deck into a live, interactive web application
- Next.js page routing and React component architecture
- Scroll-based animations using Intersection Observer API
- Responsive design that works on desktop and mobile
- One-click deployment to Vercel's global edge network

## 📋 Built By

**Lovely Marshall** — Sales Engineer | Startup Acceleration & Developer Enablement

## 🛠️ Tech Stack

- **Framework**: Next.js (React)
- **Language**: JavaScript/JSX
- **Styling**: CSS-in-JS (inline styles)
- **Deployment**: Vercel
- **Animations**: Intersection Observer API + CSS transitions

## 📝 What I Learned

1. **Next.js Pages Router** — Each file in `/pages` becomes a route automatically
2. **React Components** — Breaking a presentation into reusable slide components
3. **useState & useEffect** — Managing which slide is active and triggering animations
4. **Intersection Observer** — Detecting when slides scroll into view without performance overhead
5. **Vercel Deployment** — `git push` triggers automatic builds and deploys globally

## 🏗️ How to Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
